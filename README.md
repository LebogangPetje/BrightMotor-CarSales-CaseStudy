Introduction

This project analyzes the Bright Car Sales dataset, which contains daily vehicle sales and pricing information. As a Junior Data Analyst, the goal is to extract insights that support the new Head of Sales in improving performance, expanding the dealership network, and optimizing inventory.

Objectives

This analysis focuses on:

Identifying car makes and models that generate the most revenue

Examining the relationship between price, mileage, and year

Determining which regions have the highest sales volumes

Highlighting emerging customer purchasing trends

Providing recommendations to improve profitability and efficiency

Tools Used

SQL platforms: Snowflake, SQL Server, BigQuery, MySQL

Data visualization: Excel, Power BI, Looker Studio

Presentation tools: PowerPoint, Canva

Planning tools: Miro, Figma

Tasks Overview

Planning (Miro): Data flow, key insights, and calculations

Data Processing (SQL): Load CSV, clean data, create metrics such as revenue and profit margin

Analysis & Visualization: Build dashboards and explore trends

Presentation: Summarize insights and recommendations
